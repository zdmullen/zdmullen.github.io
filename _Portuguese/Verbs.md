# This page details the conjugation of different types of Portuguese verbs.



| Verb conjugation  | falar     |   |   |   |
|-------------------|:-----------:|---|---|---|
| eu                | falo      |   |   |   |
| tu                | falas     |   |   |   |
| ela, ele, você    | fala      |   |   |   |
| nós               | falamos   |   |   |   |
| eles, elas, vocês | falam     |   |   |   |
